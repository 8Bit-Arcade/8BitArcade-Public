## What to Include in Support Requests

### For Technical Issues

Please provide:
1. **Wallet address** (for account lookup)
2. **Transaction hash** (if relevant)
3. **Browser and version** (e.g., Chrome 120)
4. **Device** (Windows/Mac/Mobile)
5. **Exact error message** (screenshot helpful)
6. **Steps to reproduce**
7. **When it started**

### For Reward Issues

Please provide:
1. **Your wallet address**
2. **Date and time** of issue
3. **Game name**
4. **Your rank** (screenshot of final leaderboard)
5. **Expected vs actual reward**
6. **Transaction hash** (check Arbiscan)
